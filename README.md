# ShootThisMap
## Description
A third-person shooter that uses the main subsystems of Unreal Engine: Blueprints, Animation Graphs, Behavior Tree, Niagara, UMG.

- "Tab" - Change weapon
- "RMB" - Zoom for rifle
- "Esc" - Pause
